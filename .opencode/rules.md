# BasketIndex Project Rules

## Typography: Always Regular Dash (-), Never Em-Dash (—)

When writing any text — code comments, documentation, landing pages, commit messages, UI copy, HTML, CSS, JavaScript — always use the regular hyphen-minus character (U+002D, `-`, the standard dash key) **never** the em-dash character (U+2014, `—`).

This applies to:
- All languages: English, Bulgarian, German, etc.
- All file types: `.js`, `.html`, `.md`, `.css`, `.json`, `.txt`
- All projects in the BasketIndex ecosystem
- The `basketindex-landing` project
- All future files

If existing text contains em-dashes, replace them with regular dashes.
